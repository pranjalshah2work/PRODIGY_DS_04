# PRODIGY_DS_04

## Task Objective
Analyze and visualize sentiment patterns in social media data to understand public opinion and attitudes towards specific topics or brands.

## Tools & Technologies Used
- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- TextBlob
- GitHub

## Dataset Used
Twitter Sentiment Dataset

## Steps Performed
1. Loaded and explored the dataset
2. Renamed dataset columns for better understanding
3. Analyzed sentiment categories
4. Visualized sentiment distribution using charts
5. Interpreted public opinion trends

## Visualizations Created
- Sentiment Distribution Bar Chart
- Sentiment Percentage Pie Chart

## Key Insights
- Negative sentiments were highest in the dataset
- Positive sentiments were also significantly high
- Neutral and irrelevant sentiments formed a smaller portion of the data

## Outcome
Successfully analyzed social media sentiment patterns and visualized public opinion using Python.
